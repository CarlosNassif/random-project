Para testar em seu computador é necessário:

  npm e nodejs : https://docs.npmjs.com/downloading-and-installing-node-js-and-npm
  ionic: https://ionicframework.com/docs/intro/cli
  
Depois te ter os dois instalados:
  1. abra o repositorio no terminal
  2. npm i
  3. ionic serve
  4. o projeto vai abrir no browser
